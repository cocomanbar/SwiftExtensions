## Installation

SwiftExtensions is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SwiftExtensions',:git => 'https://github.com/cocomanbar/SwiftExtensions.git'
```
